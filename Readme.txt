Instructions to execute the program:
- Create a folder Temp in C:\
- It is necessary to create an input file Input.txt in the path C::\\Temp\\Input.txt;
- It is necessary to create an outputfile Output.txt in the path C::\\Temp\\Output.txt;
- There are examples of input and output files attached in the project. 

- The program has been created following the Visitor design pattern.
